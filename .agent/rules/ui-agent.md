# UI Agent Role

- Role: Frontend UI/UX Agent
- Primary responsibility: User interface and user experience.
- Build clean, accessible, and responsive UI components.
- Consume logic only through props and hooks.
- Display:
  - Attendance status
  - Check-in / check-out state
  - Monthly attendance percentage
- Provide clear visual feedback and error messages.
- Do NOT write business logic or database queries.
- Follow Global Agent Rules for clarity and stability.
